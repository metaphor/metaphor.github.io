Presenting Code

May 17th, 2013

Pain Point

As a software developer going to give presentations, the frequent requirement is to present code, retaining the syntax highlight. Your editor has already highlighted your code, but if you copy & paste the code directly into to presentation (PowerPoint or Keynote), the code might be left default color.

This post is going to show you a way satisfy yourself.

Solution

The Silver Bullet underneath is RTF.

Microsoft PowerPoint & Mac OS Keynote both support displaying RTF text.

By default, your copy is plain text and has nothing to do with format, therefore PowerPoint or Keynote render it as plain text as well.

That means if your editors could copy your code as RTF, then paste the result into presentation, it will display syntax highlighted. The question left to us is how to copy code as RTF in your code editors. This is is just a simple conversion, from your editor syntax highlight style to RTF.

Reference Implementation

Easy Enough!

Some guy might find that, Eclipse can easily do this(try it!), that’s because it builds in the functionality, while other editors might not be so obvious, such as IntelliJ. But the support could be done by certain plugin.

For this post, I prefer Sublime Text 2, since it is easy to setup and it supports a lot of languages syntax highlights. (you could copy the code to Sublime Text 2 first from other editors) Download & Install Sublime Text 2

  • Install Package Manager

To install package, you need the package manager for convienence, please following http://wbond.net/sublime_packages/package_control/installation

  • Install SublimeHighlight

SublimeHighlight is the right plugin to enable copy as RTF.

For how to install package, please following

1. Command + Shift + P to popup the control panel (on Windows it might be Ctrl + Shift + P)
2. choose "Install Package"
3. choose "SublimeHighlight" in the package list

For plugin configuration detail following https://github.com/n1k0/SublimeHighlight

  • Copy as RTF

First you need to set the right syntax highlight to your code, and then select the region you are copying, on right context menu “Copy as RTF”, paste the code to your Keynote or PowerPoint(you might need select keep formatting on PowerPoint).
